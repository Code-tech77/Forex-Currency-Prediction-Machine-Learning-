# 💱 Forex Currency Prediction : End-to-End Machine Learning & LightGBM

## 🚀 Overview
This project is part of **Week 9** of the Microsoft-affiliated Data Science Internship.  
The goal is to build an **end to end machine learning pipeline** to analyze and predict Forex currency movements using real world financial data and the powerful **LightGBM algorithm**.

---

## 🎯 Objectives

    Build a complete end to end machine learning pipeline
    Analyze Forex currency trends and patterns
    Predict future currency price movements
    Apply LightGBM for high performance modelling

---

## 🧠 Key Concepts

💱 End-to-End Machine Learning

A full ML workflow covering data collection, preprocessing, feature engineering, model training, evaluation, and prediction.


💱 Forex (Foreign Exchange)

The global market for trading currencies, where price movements are influenced by economic and market factors.


💱 LightGBM (Light Gradient Boosting Machine)

A high-performance gradient boosting framework that uses decision trees and is optimized for speed and accuracy on large datasets.

## ⚙️ Project Workflow

### 1. Data Collection & Cleaning

    Load historical Forex data
    Handle missing values
    Format and structure time series data

### 2. Exploratory Data Analysis (EDA)

    Analyze price trends and volatility
    Visualize currency movement patterns
    Identify key behaviours in the data

### 3. Feature Engineering

    Create technical indicators (moving averages, returns)
    Generate lag features for time dependency
    Normalize/scale data for modelling

### 4. Model Building (LightGBM)

    Train LightGBM model on structured data
    Optimize using boosting techniques
    Improve performance with parameter tuning

### 5. Model Evaluation

    Evaluate using metrics (e.g., RMSE, MAE, accuracy)
    Compare predicted vs actual values
    Analyze model performance
